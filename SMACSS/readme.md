
# SMACSS — Scalable and Modular Architecture for CSS


## ¿Para qué sirve?
SMACSS organiza todo el CSS en categorías claras, creando proyectos escalables.
Es ideal para equipos grandes o aplicaciones de larga duración.


## ¿Cuándo escoger SMACSS?
- Cuando el proyecto crecerá con el tiempo.
- Cuando hay muchos componentes y módulos.
- Cuando se necesita orden y separación estricta.


## Organización del proyecto
- base/ → reglas globales
- layout/ → estructuras principales
- modules/ → componentes reutilizables
- state/ → variaciones temporales
- theme/ → estilos alternativos


## Lección aprendida
SMACSS permite mantener un proyecto grande sin que el CSS se vuelva inmanejable.
