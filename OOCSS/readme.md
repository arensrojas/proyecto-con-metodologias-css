# OOCSS — Object Oriented CSS


## ¿Para qué sirve?
OOCSS separa el CSS en dos categorías: estructura (layout) y apariencia (skin).
Permite reutilizar componentes fácilmente.


## ¿Cuándo escoger OOCSS?
- Cuando se busca máxima reutilización.
- Cuando el diseño es repetitivo.
- Cuando el equipo quiere separar completamente estructura y visual.


## Organización del proyecto
- layout.css → estructuras
- skin.css → colores, estilos visuales
- utilities.css → clases genéricas reusables


## Lección aprendida
OOCSS enseña a pensar en componentes reutilizables, evitando duplicar CSS y mejorando consistencia.
