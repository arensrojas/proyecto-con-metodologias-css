# BEM — Block, Element, Modifier


## ¿Para qué sirve?
BEM es una metodología que ordena el CSS en bloques independientes, elementos internos y modificadores.
Es ideal para mantener proyectos escalables, legibles y reutilizables.


## ¿Cuándo escoger BEM?
- Cuando el proyecto es mediano o grande.
- Cuando se trabaja con componentes (React, Vue, Angular).
- Cuando se necesita mantener claridad en clases.


## Organización del proyecto
- Cada componente tiene su propio bloque.
- Los elementos internos se representan con `__`.
- Los modificadores se representan con `--`.


## Lección aprendida
BEM ayuda a escribir CSS más ordenado, modular y mantenible sin depender de frameworks.
